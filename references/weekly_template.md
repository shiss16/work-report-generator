# 周报模板

## 格式规范

```markdown
## YYYY-MM-DD - YYYY-MM-DD 工作内容报告

> 会议 N 场 | 有纪要 N 场 | 任务 N 个

### 📊 概览

| 指标 | 数值 |
|------|------|
| 会议总数 | N 场 |
| 有纪要 | N 场 |
| 无记录 | N 场 |
| 最忙日 | MM月DD日（N 场）|
| 完成任务 | N 个 |
| 待办任务 | N 个 |

---

### 📅 MM月DD日（周X）

**会议标题** ✅
  > HH:MM-HH:MM
  > 纪要摘要，不超过 150 字

**完成：**
  - 任务标题 [项目名]

---

### 📊 工作脉络总结

1. **工作主线 1** — 本周完成了XX，推进到YY阶段。
2. **工作主线 2** — 简述。
3. **工作主线 3** — 简述。

> ⚠️ 注：N 场会议纪要获取失败，N 场需额外授权。
```

## 状态图标

| 图标 | 含义 |
|:----:|------|
| ✅ | 有纪要，获取成功 |
| ⚠️ | 有纪要，获取失败 |
| 🔒 | 需授权 |
| — | 无纪要/录制 |

## 渲染规则

- 标题：`## YYYY-MM-DD - YYYY-MM-DD 工作内容报告`
- 数据来源行：`> 会议 N 场 | 有纪要 N 场 | 任务 N 个`
- 概览用表格，每日详情用分隔线 `---`
- 会议用加粗标题 + 状态图标 + 引用块
- 工作脉络总结 3-5 条，每条 1-2 句话
- 目标长度：一周会议 ≤ 15 场时控制在 50-70 行以内
