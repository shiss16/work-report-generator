# 日报模板

## 格式规范

```markdown
## YYYY-MM-DD 工作日报

> 会议 N 场 | 有纪要 N 场 | 待办 N 个

### YYYY-MM-DD（周X）

✅ **会议标题** (HH:MM-HH:MM)
  纪要摘要内容，不超过 100 字

✅ 完成
  - 任务标题 [项目名]

📌 待办
  - 任务标题 [项目名]

⚠️ 僵尸任务
  - 任务标题 — 已搁置 N 天
```

## 状态图标

| 图标 | 含义 |
|:----:|------|
| ✅ | 有纪要，获取成功 |
| ⚠️ | 有纪要，获取失败 |
| 🔒 | 需授权 |
| — | 无纪要/录制 |

## 渲染规则

- 标题：`## YYYY-MM-DD 工作日报`
- 每日标题：`### YYYY-MM-DD（周X）`
- 会议用加粗标题 + 时间范围 + 纪要摘要
- 会议中的待办项用 `- [ ]` 格式
- 完成/待办用简单列表
- 无内容时显示"（暂无）"
- 目标长度：当日会议 ≤ 3 场时控制在 30 行以内
